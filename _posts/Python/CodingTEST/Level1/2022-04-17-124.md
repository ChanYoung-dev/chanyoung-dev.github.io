# 124 나의 숫자

진법/진수의 문제 유형은 두가지 경우가 있다
1. 10진수 0, 1, 2 .. ,N 과 같이 N진법에 대응하는 10진수 숫자가 0부터 시작하는 경우
2. 10진수 1, 2, 3 .. ,N 과 같이 N진법에 대응하는 10진수 숫자가 1부터 시작하는 경우

- ### 정답
```python
def solution(n):
    if n != 0:
        q, r = divmod(n - 1, 3)
        return solution(q) + '124'[r]
    return ""
```
